# Ashfall: Survival Command (3D Mobil) - GDD Ozet

## Vizyon
Post-apokaliptik bir savas bolgesinde, izometrik/ucuncu sahis kamerayla
dalgalar halinde gelen dusmanlara karsi hayatta kalma. Oyun, hizli aksiyon,
stratejik gelisim ve uzun vadeli ilerleme etrafinda kurulur.

## Core Loop
Misyon secimi -> Savas (dalga) -> Kaynak toplama -> Guncelleme -> Yeni dalga

## Kamera ve Hissiyat
- Izometrik 3D kamera (semi-realistic, stylized shader)
- Kisa sureli hizli aksiyon, okunabilir silhouette
- Mobil odakli UI (tek elle kontrol + yetenek butonu)

## Oynanis
- Auto-aim + hareket kontrolu
- Dalgalar ilerledikce dusman yogunlugu ve cesidi artar
- Boss dalgalari (5. dalga ve katlari)

## Karakter Sistemi
Rarite: Common / Rare / Epic / Legendary
- Her karakterin aktif + pasif yetenegi var
- Oynadikca mastery kazanir

## Silah Sistemi
Silahlar seviye 1-20+
Modifikatorler: fire / poison / crit / splash

## Ekonomi
Kredi: dusman, misyon, etkinlik
Kredi ile:
- Karakter acma
- Silah upgrade
- Yetenek gelistirme

## Menuler (Farkli UI Yaklasimi)
Klasik buton listesi yerine **Komuta Merkezi** temasi:
- Sol panel: Karakter (silah + perk)
- Sag panel: Gorevler + gunluk
- Alt panel: Savas hazirligi (start + ekipman)

## Uzun Vadeli Gelisim
- Player level
- Skill tree (damage/health/speed)
- Character mastery
- Sezonluk hedefler (cosmetic odul)
