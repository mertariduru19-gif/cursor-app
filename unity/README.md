# Unity 3D Mobil Proje (Icerik)

Bu klasor Unity 3D mobil oyun iskeletini barindirir.

## Klasorler
- Assets/Scripts/Core: profil, ekonomi, bootstrap
- Assets/Scripts/Combat: silah, dusman, dalga
- Assets/Scripts/Player: oyuncu kontrolu
- Assets/Scripts/Input: mobil joystick
- Assets/Scripts/Camera: izometrik kamera
- Assets/Scripts/UI: HUD ve menu

## Not
Unity Editor icinde sahne ve prefab kurulumlari gerekir.
